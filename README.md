# [初见的诗篇](https://ysgcdq.github.io/)

爱情常常始于某个不经意的瞬间，像午后阳光穿过树叶的缝隙，突然落在心上最柔软的地方。

那可能是一个回眸，一次指尖的轻触，或是深夜时分隔着屏幕的一句“你睡了吗”。初见的爱情是清晨的露珠，晶莹剔透，还未沾染尘世的重量。你会突然注意到他说话时微微上扬的嘴角，她思考问题时轻蹙的眉头，或者两人意外相似的歌单。

科学家说这是多巴胺的魔法，诗人称之为命运的序章。而身处其中的人，只觉得自己变成了一个收集者——收集对方每一次微笑的模样，每句看似随意却深藏温柔的关心，每个让心跳漏掉半拍的眼神交汇。

这样的爱情像一场细雨，你不知道它何时开始落下，待你察觉时，衣衫已微湿，空气中满是青草与泥土的芬芳。一切都尚未定义，一切都充满可能。而这段朦胧时光里最珍贵的，是那份小心翼翼的靠近，是那本还未开始但已写满憧憬的诗集的第一页空白。
